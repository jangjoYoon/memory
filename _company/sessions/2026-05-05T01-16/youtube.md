## 🚀 구독자 1만 도달을 위한 플레이리스트 기획서 3가지

제이코프의 핵심 정체성인 **'시스템 설계자(System Designer)' 포지셔닝**과 **'오렌지색 강조 원칙'**을 극대화하여, 시청자가 단순한 팁을 넘어 '수익 시스템 구축 방법론'을 배우고 싶게 만드는 데 초점을 맞춥니다.

---

### 🥇 플레이리스트 1: [시스템 해부] 수익 엔진의 숨겨진 구조 (권위 확보 집중)

**🎯 핵심 목표:** 채널의 **'시스템 설계자' 권위**를 가장 강력하게 각인시키고, 시청자가 이 시스템을 이해해야만 돈을 벌 수 있다는 인식을 심어주어 높은 전환율(CTR 및 리드 마그넷 다운로드)을 유도합니다.

| 항목 | 상세 내용 |
| :--- | :--- |
| **영상 제목 (최종 확정)** | **[엔진 해부] 플레이리스트 수익화 엔진의 숨겨진 Profit Path 구조 (오렌지색 시스템 설계)** |
| **핵심 후크 (Hook)** | "당신이 아는 플레이리스트 제작법은 90%만 알고 있습니다. 나머지 10%가 당신을 월 1,000만원으로 이끌 겁니다. 지금부터 그 숨겨진 '수익 엔진'의 설계도를 해부합니다." |
| **구조 및 내용** | 1. **도입:** 왜 일반적인 방법이 실패하는가? (문제 제기) / 2. **핵심:** Profit Path(수익 경로) 시각화 원칙 설명 (오렌지색 강조의 의미) / 3. **엔진 해부:** Input → Process → Output 모듈별 수익 지점 분석 / 4. **결론:** 시스템 구축 로드맵 제시 및 다음 단계 안내 |
| **시청자 유입 전략** | **권위 기반 유입:** '시스템', '엔진', '구조'와 같은 단어를 사용하여 전문성을 강조합니다. 경쟁 채널의 단순 팁 영상보다 훨씬 깊은 통찰을 제공하여 시청 지속률(Retention) 극대화. |
| **CTA (Call to Action)** | "이 시스템을 직접 구축하고 싶다면? → [AI 기반 수익 파이프라인 설계 템플릿] 다운로드" (리드 마그넷 연결) |

### 🥈 플레이리스트 2: [경쟁 분석] 성공하는 크리에이터들의 ROI와 가격 전략 비교 (실용성 및 데이터 집중)

**🎯 핵심 목표:** 시장 경쟁 환경에 대한 **데이터 기반 통찰력**을 제공하여, 시청자가 자신의 비즈니스에 즉시 적용 가능한 구체적인 '가격 책정'과 'ROI 분석' 능력을 갖추도록 돕습니다.

| 항목 | 상세 내용 |
| :--- | :--- |
| **영상 제목 (최종 확정)** | **[경쟁 ROI] 성공하는 플레이리스트 크리에이터들의 숨겨진 가격 전략 및 수익률 비교 분석** |
| **핵심 후크 (Hook)** | "당신이 생각하는 마진은 틀렸습니다. 실제 돈을 버는 플레이리스트 크리에이터들이 사용하는 '숨겨진 가격 책정 공식'과 그들이 얻는 실질적인 ROI를 데이터로 비교 분석합니다." |
| **구조 및 내용** | 1. **도입:** 왜 가격 전략이 수익의 핵심인가? (문제 제기) / 2. **분석:** 경쟁사 A, B, C의 Pricing Model(구독/템플릿/서비스) 해부 / 3. **비교:** 각 모델별 예상 ROI 및 마진율 시뮬레이션 (데이터 제시) / 4. **전략:** 내 시스템에 적용할 수 있는 가격 책정 프레임워크 제시 |
| **시청자 유입 전략** | **실용성 기반 유입:** 'ROI', '가격 전략', '비교 분석' 등 구체적인 비즈니스 용어를 사용하여, 시청자가 즉각적인 금전적 이득을 기대하게 만듭니다. 데이터와 비교를 통해 신뢰도를 높입니다. |
| **CTA (Call to Action)** | "나만의 수익화 시스템 구축 가이드라인 전체 보기" (Tier 2 상품 연결) |

### 🥉 플레이리스트 3: [자동화 실습] AI 기반 수익 파이프라인 설계 템플릿 구현 (실행 및 자동화 집중)

**🎯 핵심 목표:** 이론을 실제 **구현 가능한 결과물(템플릿)**로 전환하여, 시청자가 복잡한 시스템 구축 과정을 자동화하고 즉시 실행할 수 있도록 지원합니다.

| 항목 | 상세 내용 |
| :--- | :--- |
| **영상 제목 (최종 확정)** | **[AI 실습] 5분 만에 끝내는 AI 기반 수익 파이프라인 설계 템플릿 완벽 구현 가이드** |
| **핵심 후크 (Hook)** | "복잡한 시스템 설계는 이제 그만! 이 영상에서 제공하는 'AI 수익 파이프라인 설계 템플릿'을 따라 하면, 당신도 오늘 당장 자동화된 수익 엔진을 구축할 수 있습니다. (실습 포함)" |
| **구조 및 내용** | 1. **준비:** 필요한 AI 도구 및 데이터 정제 환경 설정 / 2. **템플릿 공개:** 핵심 모듈(Input/Process/Output)의 실제 템플릿 시연 / 3. **자동화 실습:** 템플릿을 활용하여 수익 파이프라인 자동 설계 과정 단계별 실행 / 4. **마무리:** 시스템 적용 및 다음 확장 로드맵 제시 |
| **시청자 유입 전략** | **실행 기반 유입:** 'AI', '템플릿', '실습' 키워드를 전면에 내세워, 시청자가 지루한 이론 대신 즉각적인 결과물을 얻고 싶어 하게 만듭니다. (가장 높은 실질적 가치 제공) |
| **CTA (Call to Action)** | "템플릿 전체 파일 및 추가 자동화 팁 받기" (리드 마그넷 연결) |

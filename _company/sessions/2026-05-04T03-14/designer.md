## 💡 리드 마그넷 Reels 썸네일 디자인 시안 (1/3)

이 시안은 'AI 기반 수익 파이프라인 설계 템플릿'의 권위와 시스템적 가치를 오렌지색을 통해 극대화하여 잠재 고객의 즉각적인 클릭을 유도하도록 설계되었습니다. [The Blueprint]와 [The Engine] 컨셉을 통합하여 전문성을 시각화합니다.

### 🎯 핵심 디자인 목표
*   **주요 메시지:** 시스템(Blueprint)이 수익(Engine)으로 전환된다는 명확한 인과관계 제시.
*   **강조 색상:** 오렌지 (권위, 수익 지점 강조).
*   **스타일:** 대담하고 시각적으로 충격적이며, 전문적인 느낌.

---

### 🖼️ 시안 1: [The Blueprint & Engine 통합형] (권위 극대화)

이 시안은 시스템의 구조와 결과물을 동시에 보여주어 '설계자'로서의 권위를 가장 강력하게 어필합니다.

**레이아웃:** 분할 구도 (좌/우 또는 상/하)
**핵심 요소:** Blueprint (구조) + Engine (결과)

| 요소 | 내용 및 디자인 지침 | 색상 코드 |
| :--- | :--- | :--- |
| **배경** | 짙은 차콜 그레이 또는 블랙 (전문성 확보) | `#1A1A1A` |
| **Blueprint 영역 (좌측/상단)** | 시스템의 구조를 상징하는 추상적인 흐름도나 설계 도면의 일부. 미니멀하게 처리. | `#333333` |
| **Engine 영역 (우측/하단)** | 수익 또는 결과물을 상징하는 역동적인 화살표, 상승 그래프, 또는 금화 모양의 아이콘 배치. | **`#FF6B00` (강조 오렌지)** |
| **타이포그래피 (헤드라인)** | "AI 수익 파이프라인 설계" (가장 크게) | `#FFFFFF` (흰색) |
| **서브 텍스트** | "시스템으로 돈 버는 법" (Blueprint 아래에 작게 배치) | `#FF9800` (밝은 오렌지) |
| **강조 효과** | Blueprint와 Engine 사이를 연결하는 화살표나 선을 강렬한 오렌지색으로 처리하여 '전환'의 시각적 흐름을 강조. | **`#FF6B00`** |

**시각적 설명:** 어두운 배경 위에서, 왼쪽에는 복잡하지만 정돈된 설계도(Blueprint)가 있고, 오른쪽에는 그 설계가 작동하여 수익으로 폭발하는 역동적인 에너지(Engine)가 오렌지색으로 표현됩니다. 시청자는 이 두 요소 사이의 연결고리(템플릿)를 얻어야 한다는 것을 직관적으로 인지합니다.

---

### 🖼️ 시안 2: [The Engine 집중형] (결과 중심 강조)

이 시안은 최종 결과물인 '수익'에 초점을 맞춰 즉각적인 금전적 이득을 약속합니다.

**레이아웃:** 중앙 집중형
**핵심 요소:** 오렌지색의 폭발력

| 요소 | 내용 및 디자인 지침 | 색상 코드 |
| :--- | :--- | :--- |
| **배경** | 밝은 회색 또는 미묘한 그라데이션 (오렌지를 돋보이게 함) | `#F5F5F5` |
| **중앙 이미지** | 거대한 오렌지색의 상승 그래프, 화살표, 또는 돈다발 아이콘을 중앙에 배치. | **`#FF6B00`** |
| **텍스트 (헤드라인)** | "당신의 AI 수익 엔진" (가장 크게) | `#1A1A1A` (검은색) |
| **서브 텍스트** | "시스템 설계로 자동 수익화" (작게 하단에 배치) | `#333333` |
| **강조 효과** | 전체 디자인의 70% 이상을 오렌지 계열로 채워 시각적 충격을 극대화. Blueprint 요소는 최소화하고 Engine의 폭발력만 강조. | **`#FF6B00`** |

**시각적 설명:** 배경은 밝지만, 중앙에 강력한 오렌지색 에너지(Engine)가 터져 나오는 듯한 느낌을 줍니다. 이는 '이것을 통해 돈을 벌 수 있다'는 직접적인 메시지를 전달하며, 가장 자극적이고 시선을 사로잡는 효과를 노립니다.

---

### 🖼️ 시안 3: [Blueprint 강조형] (시스템 신뢰도 중심)

이 시안은 시스템의 깊이와 논리적 구조에 중점을 두어, 잠재 고객에게 '깊이 있는 지식'을 제공한다는 인상을 심어줍니다.

**레이아웃:** 수직 분할 및 그리드 기반
**핵심 요소:** 명확한 구조와 타이포그래피

| 요소 | 내용 및 디자인 지침 | 색상 코드 |
| :--- | :--- | :--- |
| **배경** | 깨끗한 화이트 또는 아주 옅은 그레이 | `#FFFFFF` |
| **Blueprint 영역** | 깔끔하게 정리된 박스 형태의 흐름도(Flowchart)를 중앙에 배치. 선과 도형은 검정색으로 명확히 구분. | `#000000` |
| **Engine 강조** | Blueprint 내에서 수익 지점이나 핵심 모듈에만 오렌지색을 포인트로 사용. (예: 최종 결과 박스만 오렌지) | **`#FF6B00`** |
| **타이포그래피 (헤드라인)** | "AI 시스템 설계의 비밀" (권위 있는 폰트 사용) | `#1A1A1A` |
| **서브 텍스트** | "수익 파이프라인 구축 로드맵" (Blueprint 아래에 배치) | `#666666` |

**시각적 설명:** 깔끔하고 전문적인 느낌을 주며, 복잡한 시스템(Blueprint)이 얼마나 논리적으로 설계되었는지를 보여줍니다. 오렌지색은 이 논리 구조 내에서 '가장 중요한 수익 지점'만을 콕 집어 강조하여 신뢰도를 높입니다.

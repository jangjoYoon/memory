## 🌟 브랜드 비주얼 시스템 확정 가이드라인 (오렌지 강조 원칙 적용)

CEO의 지시에 따라, 수익화 전략에서 강조된 '오렌지색'을 핵심 요소로 사용하여 다음 콘텐츠 시리즈에 적용할 통일된 브랜드 비주얼 가이드라인(컬러 팔레트, 타이포그래피 스타일)을 확정합니다. 이 시스템은 **'시스템 설계자'로서의 권위**와 **'수익 지점'**을 시각적으로 명확히 전달하는 것을 목표로 합니다.

### 1. 컬러 팔레트 (Color Palette)

오렌지색을 핵심 액센트(Accent)로 설정하고, 신뢰감과 전문성을 더할 수 있는 차분한 색상을 보조색으로 사용합니다.

| 역할 | 색상 이름 | HEX 코드 | RGB 값 | 용도 및 의미 |
| :--- | :--- | :--- | :--- | :--- |
| **Primary Accent (핵심 강조)** | **System Orange** | `#FF8C00` | (255, 140, 0) | 수익 지점, 핵심 로직, 시스템 흐름, 행동 유도(CTA). **가장 강력하게 사용.** |
| Secondary Base (신뢰/전문성) | Deep Navy | `#0A1931` | (10, 25, 49) | 배경, 본문 텍스트, 전체적인 권위와 깊이를 부여. |
| Tertiary Neutral (배경/안정감) | Light Gray | `#F4F4F9` | (244, 244, 249) | 여백, 카드 배경, 시각적 안정감을 제공하여 오렌지색을 돋보이게 함. |
| Tertiary Accent (보조 강조) | Soft Gold | `#FFD700` | (255, 215, 0) | 중요 데이터 포인트, 성공 지표(KPI), 프리미엄 요소 강조. |

### 2. 타이포그래피 스타일 (Typography Style)

시스템의 구조와 명확성을 강조하기 위해 산세리프 계열 중 기하학적이고 현대적인 폰트를 사용하여 전문성과 기술적 정확성을 표현합니다.

| 역할 | 폰트 스타일 | 예시 폰트 | 용도 및 의미 |
| :--- | :--- | :--- | :--- |
| **Headlines (제목/후크)** | Bold, Geometric Sans-Serif | Montserrat (Bold/ExtraBold) 또는 Poppins (Black) | 시각적 충격과 권위를 전달하며, 핵심 메시지를 즉시 인식하게 함. |
| **Body Text (본문/설명)** | Clean, Highly Readable Sans-Serif | Inter 또는 Roboto | 시스템의 복잡한 로직을 명확하고 오류 없이 전달하여 신뢰도를 높임. |
| **System Flow (흐름 강조)** | Monospace / Code Font | Fira Code 또는 IBM Plex Mono | 프롬프트, 데이터 흐름, 코드 예시 등 기술적인 세부 사항에 사용되어 전문성을 극대화함. |

### 3. 디자인 가이드라인 요약 및 적용 원칙

1.  **오렌지색의 역할:** 오렌지색(`#FF8C00`)은 **'시스템이 작동하는 지점', '수익을 창출하는 로직', '핵심 전환점'**에만 전략적으로 사용되어야 합니다. 배경이나 불필요한 요소에 사용하지 않아 시각적 피로도를 줄이고 강조 효과를 극대화합니다.
2.  **구조의 명확성:** Deep Navy와 Light Gray를 주 색상으로 사용하여, 복잡한 AI 시스템 정보가 오렌지색 액센트를 통해 **명확하게 계층화**되도록 설계합니다.
3.  **기술적 권위:** Monospace 폰트 사용을 통해 콘텐츠가 단순한 정보 제공이 아닌, **'구축된 시스템의 청사진'**임을 시각적으로 증명합니다.
